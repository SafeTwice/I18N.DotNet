string Localize ( FormattableString  frmtText)

Localizes an interpolated string.

Converts the composite format string of the language-neutral formattable string frmtText ( e.g.an interpolated string) to its corresponding language-specific localized composite format value, and then generates the result by formatting the localized composite format value along with thefrmtText arguments by using the formatting conventions of the current culture.
